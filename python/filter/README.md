# 信号処理　フィルタ（LPF, HPF）のサンプル

## 概要

* LPF, HPFのサンプル  
* 乱数10000サンプルを，サンプリングレート10kHz　1秒の信号として，フィルタ処理を実行

## 実行手順

	$ python3 filter.py --help
	$ python3 filter.py
