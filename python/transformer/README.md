# Transformerの活用例を紹介

## 概要

* Transformerは自然言語処理を皮切りに時系列予測や画像認識など広い分野で活用される
* 各分野への活用例を紹介する

## 実行手順

	docker_run.shで起動後，
		http://localhost:10001
	へブラウザでアクセスし，コンソールに表示されるトークン部分をコピペしてログインする
