# Web App Environment Sample (nginx+uWSGI+Flask+Bootstrap)

## Overview

The environment sample of Web Application using nginx, uWSGI, Flask and Bootstrap.

## How to use

```
docker-compose build
docker-compose up
```

Acess to `http://localhost:5000` in your browser.
